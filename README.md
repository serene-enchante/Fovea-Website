# Fovea

The _fovea centralis_ is a part of bird's eyes which allows them to see distance objects with acute detail. This project provides a centralized and concise environment to view broad, de-centrlaized citizen science data.

**Fovea** is a data access and management interface, created to maintain open citizen science data from Audubon Christmas Bird Counts. This website allows birders to view and download their local count information. It also acts as a data entry platform for count organizers to update count records and spatial data.
