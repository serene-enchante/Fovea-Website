# Fovea

The _fovea centralis_ is a part of bird's eyes which allow them to see distance objects with acute detail. Likewise, this project provides a centralized and concise environment to view broad, de-centrlaized citizen science data. The web interface abstracts professional data schemas and organizion practices so that contributers can maintain autonomy other their data input while still enforcing scientific data validity.

**Fovea** is a data access and management interface, created to maintain open citizen science data from Audubon Christmas Bird Counts. This website allows birders to view and download their local count information. It also acts as a data entry platform for count organizers to update count records and spatial data.
