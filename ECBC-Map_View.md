# Header row 
Left: Header - title of current focus (if I am viewing zone 12 (zid=”12” in gpkg) then header reads “Zone 12”)
Right: icon buttons - copy link, download files (opens download options), edit (opens editor page for current item)

# Content row
Left 1/4 of width: List of items (such as the zones in the circle if the circle is currently being viewed) (reference list style used on the current “maps” page) (this list has an independent scrollbox)
Right 3/4 of width: Leaflet window of current gpkg.
